# Day 38 (Tuesday) -  NLP Foundations

## Description
This task covers the basics of natural language processing (NLP) by training Word2Vec models, computing cosine similarity, and exploring the polysemy of words. The task also covers the concepts of sentiment analysis and word embeddings.

## Key Learnings

- Word2Vec can be used to capture semantic relationships between words in a corpus of text.
- Cosine similarity can be used to measure the similarity between two vectors in a high-dimensional space.
- The polysemy of a word can be explored by analyzing its neighbors in the Word2Vec embedding space.
- Sentiment analysis can be performed using Word2Vec by computing the cosine similarity between a review and a sentiment-bearing word.
- Word embeddings can be used to capture the semantic meaning of words in a corpus of text.
